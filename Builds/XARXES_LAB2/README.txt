Instrucciones de uso:

·El proyecto contiene los ficheros UDP/TCP desarrollados para la práctica, además de dos carpetas que contienen los .exe

·También contiene 2 escenas (Server/Client) en el cual el gameObject contiene los componentes de script correspondientes
para cada tipo de conexión (TCP - by default / UDP - disabled script).

·De las anteriores escenas existen dos apps TCPServer y TCPClient, para su correspondiente uso se debe inicializar primero
el servidor y posteriormente el cliente.
